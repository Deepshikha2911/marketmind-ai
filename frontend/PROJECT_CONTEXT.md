# MarketMind AI

MarketMind AI is an AI-powered marketing analytics platform.

## Tech Stack

Frontend
- Next.js 16
- TypeScript
- Tailwind CSS
- App Router

Backend
- FastAPI
- REST APIs

The backend is already completed.

The frontend should only consume these APIs.

## Features

- Dashboard
- Dataset Upload
- Revenue Prediction
- AI Insights
- Budget Optimizer
- Revenue Forecast
- Scenario Simulator
- Complete Analysis
